# Objectives

The main objective of *MyErasmusJourney* is to develop a web platform that connects current and future Erasmus students, allowing them to share experiences, access useful information, and build a community around international mobility programs.

The application aims to provide both social interaction and informative content related to Erasmus destinations across Europe.

---

## 🎯 Functional Objectives

The functional objectives of the application are:

- Provide future Erasmus students with reliable information about countries, cities, and universities to help them choose their destination.
- Allow students to publish and share their Erasmus experiences, including stories, advice, and personal insights.
- Enable interaction between users through comments and engagement with published posts.
- Facilitate connections between students who are or will be located in the same city or country.
- Support the creation and discovery of social events organized in Erasmus destinations.
- Help students solve common problems before and during their Erasmus mobility by sharing knowledge and experiences.
- Allow users to explore destinations based on real experiences shared by other students.

---

## ⚙️ Technical Objectives

From a technical perspective, the application aims to:

- Develop a full-stack web application with a clear separation between frontend and backend.
- Implement a REST API to manage users, posts, destinations, and interactions.
- Design and implement a relational database to store structured information about users and content.
- Ensure secure authentication and authorization mechanisms for different user roles.
- Support image uploading for user profiles and published posts.
- Integrate external services such as maps or location-based APIs (e.g., Google Maps).
- Implement a scalable architecture that allows future integration of additional features such as recommendations or notifications.

---

## 📌 Summary

This project focuses on building both a social and informative platform for Erasmus students, combining community interaction with useful data about international mobility experiences. The objectives defined above establish the scope of the system before the implementation phase begins.
